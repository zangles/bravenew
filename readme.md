## Find Your Service

Instalation

1. Create database
2. Create a .env file coping .env.example file
3. Config database settings in .env file
4. Config Google place API key in .env file 
5. ``composer install``
6. ``php artisan migrate --seed``
7. ``php artisan key:generate``
